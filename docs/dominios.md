Cada dominio tiene al menos 3–4 entidades relacionales naturales.

**Total**: 150 dominios

## Política de uso

- **Un dominio por aprendiz por trimestre** — aplica a todos los bootcamps del mismo trimestre
- Si un aprendiz repite bootcamp en un trimestre posterior, se le asigna un dominio diferente
- Los dominios marcados con `★` están reservados para ejemplos en teoría y ejercicios guiados — no asignar a aprendices
- Para asignación automatizada usar `scripts/assign_domains.py`
